# Release Handoff Procedure

Purpose: repeatable procedure for merge-ready handoff.

## Procedure

1. Confirm branch/worktree behavior followed `../workflow.md`.
2. Run required validation from `../../CI_POLICY.md`.
3. Summarize changes, validations, risks, and any skipped checks.
4. Push branch and provide PR URL for review.
5. Keep open follow-up work in `../backlog.md` only.

## Notes

- Do not duplicate workflow or CI policy here.
- If branch protection requires base-branch sync, perform it before merge-ready handoff.
